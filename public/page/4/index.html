使用方法

brew -v 查询Homebrew版本
brew -h brew帮助
brew update 更新Homebrew
brew install 安装任意软件
brew uninstall 卸载任意软件
brew search 查询任意包
brew list 列出安装列表
brew info 查看任意包内容信息
brew upgrade 更新任意包
brew cleanup 删除具体旧软件
brew cleanup 删除所有旧软件
brew outdated 已安装的包是否需要更新
